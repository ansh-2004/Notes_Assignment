# AWS CLI Setup on Windows

- Go to google and search for aws cli install windows
- Go to Install aws cli version 2
- Simply install it using MSI Installer
- After installed, go to cmd and type **aws --version** to is it installed correctly or not 

# AWS CLI Setup on Mac
- Same steps as of windows 

# AWS CLI Setup on Linux
- curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
- unzip awscliv2.zip
- sudo ./aws/install
